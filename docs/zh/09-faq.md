# 💬 常见疑问

像跟朋友聊天那种问答。挑你关心的看就好。

## 🤖 它写的代码我看不懂怎么办

**不需要看懂。**

你的工作是判断"成果对不对"——它给你的产物（一个网页、一个工具、一个能跑的小应用），你打开看一眼，能用就是能用，不能用就告诉它哪里不对。

中间的代码是它的"草稿本"，你完全不用看。

## 😱 万一它做错了，能撤销吗

**能。**

所有改动都发生在你电脑上你的项目里，没有任何东西被推到网上。如果某次它做的事你不满意，跟它说："把刚才那次改动撤销掉。"它会帮你回到之前的状态。

> 💡 如果你的项目已经在用 git，所有撤销都会变得更安全——你随时可以回到任何一个版本。如果不熟悉 git，让 agent 帮你设置就好。

## 🛠️ 我电脑上没装过开发环境，能用吗

**能。**

第一次让它做需要某个工具（比如 Node.js、Python）的事时，它会发现"啊我需要 xxx"，然后告诉你："要我教你装一下吗？"——你说"好"，它就一步一步带你装。

你不需要自己去搜"怎么装 xxx"。

## 🔒 数据安全吗？我的内容会上传到哪里吗

**全在本地。**

- 你的项目文件，**只在你电脑上**
- agent 写出来的代码，**只在你电脑上跑**
- 你跟 agent 的对话，会发给你选的 AI 服务（这部分你装 agent 时会知道用的是哪家）

如果你担心某段对话不想发给 AI 服务，那就别把它写进意图里——agent 看不到你没说的东西。

## 📂 我可以同时开几个项目吗

可以。顶部有 tab，每个项目独立——切换不会丢进度。

## 🛑 它正在做事，我想中途打断

直接在面板里跟它说话，它会停下来听你说。完全停就按 ESC 或者输入 `/stop`。

## 🔄 我自己改了点东西，它还能继续吗

按顶部的同步按钮。会有一个 agent 去看一眼当前现状，把"会议纪要"刷一下。然后接着做。

## ⏰ 关掉应用之后再回来，进度还在吗

**还在。**

你不用做任何收尾，直接关掉就好。下次打开同一个项目，它就从你离开的地方继续。

## 🐢 它跑得有点慢

- 看看你绑的是哪个 AI 服务——有些大模型反应就是慢
- 把要做的事拆得小一点，每次只做一件
- 如果是大事，多花点时间在前面"谈方案"——一旦谈清楚，做起来就快

## 🌍 我换台电脑还能继续吗

应用本身不做云同步。但你的项目目录里已经包含了一切上下文——把它放进 git，换台电脑克隆下来打开就能继续。

## 🖥️ macOS 提示"无法验证开发者"

右键应用图标 → 打开。会出现"仍要打开"按钮。确认一次以后系统会记住。

如果提示是**"已损坏"**（不是"未识别"），在终端跑一次：

```
xattr -cr "/Applications/Bamdra Loom.app"
```

## 🪟 Windows 提示"未知发行者"

点 SmartScreen 弹窗里的"更多信息" → "仍要运行"。

## 🐧 Linux 上找不到图标

AppImage 需要 fuse 支持，部分发行版要先：

```
sudo apt install libfuse2
```

deb 包安装后图标会自动出现在系统菜单里。

## 🆙 应用怎么升级

启动后会自动检查。有新版本时顶部会出现一个下载按钮，点一下下载，下载完点重启即可。

如果自动下载失败，会弹窗提示你——可以重试，也可以直接到 GitHub releases 页手动下载新版安装包覆盖安装。
