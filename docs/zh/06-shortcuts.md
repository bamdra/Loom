# ⌨️ 快捷键

Loom 为键盘优先设计。学会几个快捷键，双手不用离开键盘。

## 理念：双手不离键盘

Loom 里的每个核心操作 —— 切换 agent、打开会话、发起任务、关闭视图、调整设置 —— 都有快捷键。目标是从意图到交付代码，全程不碰鼠标。

## 速查表

### Agent 与会话

| 操作 | macOS | Windows / Linux |
|------|-------|----------------|
| 打开 Agent 1 会话 | `⌘1` | `Ctrl+1` |
| 打开 Agent 2 会话 | `⌘2` | `Ctrl+2` |
| 打开 Agent 3 会话 | `⌘3` | `Ctrl+3` |
| 打开 Agent 4 会话 | `⌘4` | `Ctrl+4` |

### 任务与操作

| 操作 | macOS | Windows / Linux |
|------|-------|----------------|
| 打开独立任务 | `⌘⇧E` | `Ctrl+Shift+E` |

### 导航

| 操作 | macOS | Windows / Linux |
|------|-------|----------------|
| 关闭当前视图 | `⌘W` | `Ctrl+W` |
| 设置 | `⌘,` | `Ctrl+,` |
| 切换终端 | `⌘⇧H` | `Ctrl+Shift+H` |
| 切换 Agent 设置 | `⌘⇧D` | `Ctrl+Shift+D` |
| 退出（双击确认） | `⌘Q` | `Ctrl+Q` |

## 小贴士

- **⌘Q 需要双击**：第一次按显示确认提示，2 秒内再按才真正退出，防止误操作。
- **切换行为**：⌘1-4 等快捷键是 toggle —— 已经打开的，再按一次就关闭。
- **可自定义**：所有快捷键可在 `settings.toml` 的 `[shortcuts]` 段重新映射。

## 一个纯键盘的工作流

```
⌘⇧E    → 打开独立任务，输入内容，⌘Enter 提交
⌘1      → 查看分析 agent 的结果
⌘2      → 看设计 agent 的方案
⌘3      → 观察实现进度
⌘4      → 看审核结论
⌘W      → 看完了，关闭当前视图
⌘,      → 需要的话调一下设置
```
