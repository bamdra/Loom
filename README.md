# Bamdra Loom

桌面端 AI 编排工具的产物分发仓库。

源码不在此仓库；这里只发布每个版本的安装包与 `latest.json` 自动更新清单。

## 下载安装

前往 [Releases](https://github.com/bamdra/Loom/releases) 下载对应平台的安装包：

- **macOS Apple Silicon**：`Bamdra Loom_<version>_aarch64.dmg`
- **macOS Intel**：`Bamdra Loom_<version>_x64.dmg`
- **Linux x64**：`bamdra-loom_<version>_amd64.deb` 或 `.AppImage`
- **Windows x64**：`Bamdra Loom_<version>_x64-setup.exe` 或 `.msi`

## 自动更新

应用启动后 5 秒会检查本仓库 Releases 的最新版本。新版本可用时状态栏会出现提示。
